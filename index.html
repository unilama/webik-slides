Co to jest "ten" test???

                                
class StackTest extends PHPUnit_Framework_TestCase
{
    public function testPushAndPop()
    {
        $stack = array();
        $this->assertEquals(0, count($stack));
 
        array_push($stack, 'foo');
        $this->assertEquals('foo', $stack[count($stack)-1]);
        $this->assertEquals(1, count($stack));
 
        $this->assertEquals('foo', array_pop($stack));
        $this->assertEquals(0, count($stack));
    }
}

TDD

  1. Piszemy test sprawdzający dodawaną funkcjonalność. Test w tym momencie zwraca błąd.
  2. Implementujemy funkcjonalności. W tym momencie napisany test powinien się udać.
  3. Refaktoryzacja napisanego kodu, aby spełniał oczekiwane standardy.
